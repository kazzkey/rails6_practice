### versions
ruby 2.6.5 / rails 6.0.3 / postgreSQL 12.3

### メモ
- rails6(6.0.3)をインストール
```
$ gem install rails -v 6.0.3
```
- rails new
```
$ rails _6.0.3_ new rails6_practice -d postgresql
```
この時、`yarn` `webpacker`が一緒にインストールされる。
